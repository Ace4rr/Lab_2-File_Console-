# Лабораторная работа

## Описание

Данный проект представляет собой консольное приложение миниоболочку с файловыми командами, реализованную на Python в рамках **Лабораторной работы №2**

### Поддерживаемые возможности (Easy)

* **Навигация по каталогам**: cd, включая `..` и `~`.
* **Список файлов:** ls с опцией `-l` для детального отображения.
* **Вывод файла:** Вывод содержимого файла в консоль.
* **Переход в другие директории:** поддержка.
* **Копирование и переименование, перемещение файлов и каталогов:** `cp`, `mv` с поддержкой рекурсировного копирования директорий.
* **Удаление файлов и директорий** Удаление файлов и директорий с подтверждением пользователя.

## Дополнительные возможности (Medium)
* **Архивирование** Архивирование и разархивирование файлов в формате `tar`, `zip`.
* **Поиск по содержимому файлов** `grep` с опциями рекурсивного поиска.

### Формат ввода
- Вводится команда оболочки (ls, cd, cat, cp, mv, rm и тп).
- Опции команд (-l, -r и тп) задаются через пробел.
- Поддерживаются как относительные, так и абсолютные пути.
- История команд логируется.

### Алгоритм работы
Пользователь вводит команду в minishell>.
Введённая строка парсится через argparse.
Вызывается соответствующий обработчик команды из модуля commands/.
Обработчик выполняет действие.
Все команды и ошибки логируются в shell.log.
Результаты выводятся на экран.
